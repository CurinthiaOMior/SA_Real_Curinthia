Sobre a matéria de Fundamentos de Tecnologia da Informação

A matéria de Fundamentos da Tecnologia da Informação apresenta os elementos essenciais que compõem um computador e seu funcionamento. Ela aborda desde o hardware, responsável pela parte física, até os sistemas operacionais, que controlam e organizam o uso desses recursos. Também inclui o estudo dos sistemas de arquivos, que estruturam dados, e dos softwares de escritório, usados para atividades práticas. Esses conhecimentos fornecem a base para entender e utilizar a tecnologia de forma eficiente no dia a dia.

Hardware, placa-mãe, processador e memória RAM

Hardware é a parte física do computador, composta por todos os componentes que você pode tocar. A placa-mãe é o circuito principal onde todos os dispositivos se conectam e se comunicam. O processador (CPU) é o “cérebro” que executa cálculos e instruções. A memória RAM é a memória principal, usada para guardar temporariamente dados e programas em uso. Juntos, esses elementos formam a base do funcionamento do computador.

Sistema operacional

Sistema operacional é o software que gerencia o hardware e permite que o usuário interaja com o computador. Ele controla recursos, organiza tarefas e executa programas. Também oferece interface gráfica e suporte a dispositivos.

Windows – Microsoft

Desenvolvido pela Microsoft, o Windows é o sistema operacional mais usado em computadores pessoais. Ele é conhecido pela interface gráfica intuitiva, grande compatibilidade com programas e foco em uso doméstico, empresarial e educacional.

Linux – Comunidade Open Source (liderado inicialmente por Linus Torvalds)

Linux não pertence a uma empresa específica. Ele é mantido por uma comunidade global de desenvolvedores e empresas que colaboram com o código aberto. É muito usado em servidores, programação, segurança e sistemas embarcados devido à sua estabilidade e liberdade de personalização.

macOS – Apple

Criado pela Apple, o macOS é exclusivo dos computadores Mac. É conhecido pelo design elegante, alta otimização de desempenho e integração com outros dispositivos da Apple, como iPhone e iPad.

Android – Google (Alphabet)

O Android foi desenvolvido pelo Google e é atualmente o sistema operacional móvel mais usado no mundo. Ele é baseado em Linux e possui código aberto, permitindo que fabricantes como Samsung e Motorola personalizem o sistema em seus smartphones.

Sistema de arquivos, modo texto e modo gráfico

O sistema de arquivos organiza como pastas e arquivos são armazenados no disco. Em modo texto (cmd), o usuário usa comandos como dir, cd e mkdir para navegar e manipular arquivos. Já no modo gráfico (Explorer), tudo é feito visualmente por janelas, ícones e menus. Ambos acessam os mesmos dados, apenas com interfaces diferentes.

Software de escritório e exemplos

Softwares de escritório são programas usados para tarefas administrativas e produtivas do dia a dia. Exemplos: Microsoft Word, para criar textos e relatórios; Microsoft Excel, para tabelas, cálculos e gráficos; Microsoft PowerPoint, para montar apresentações visuais. Eles facilitam estudos, trabalhos e organização de informações.
