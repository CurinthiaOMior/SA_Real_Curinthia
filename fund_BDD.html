Fundamentos de Banco de Dados

Sobre a matéria de Fundamentos de Banco de Dados

O conteúdo de banco de dados aborda os principais conceitos usados para organizar, armazenar e gerenciar informações de forma estruturada. Inclui o estudo dos SGBDs, dos tipos de dados, dos modelos de representação como MER e DER, além dos principais bancos utilizados no mercado. Também envolve o uso de dicionários de dados e a criação de formulários para cadastro, simulando como as informações são registradas em sistemas reais.

Sistema de Gerenciamento de Banco de Dados(SGBD)

Um SGBD (Sistema Gerenciador de Banco de Dados) é um software usado para criar, organizar, consultar e gerenciar bancos de dados. Os principais tipos são: relacional (MySQL, PostgreSQL), não relacional (MongoDB, Redis), hierárquico, orientado a objetos e em nuvem. Ele é aplicado em sistemas comerciais, sites, aplicativos, empresas e qualquer lugar que precise armazenar informações de forma segura e estruturada.

Principais Bancos de Dados

Entre os mais utilizados estão: MySQL, PostgreSQL, Oracle, SQL Server, MariaDB, MongoDB, Firebase, Redis e SQLite. Cada um tem características específicas para diferentes tipos de aplicações, desde sites simples até grandes sistemas corporativos.

SQL Server

O SQL Server é um banco de dados relacional desenvolvido pela Microsoft, voltado para aplicações corporativas. Ele oferece alta segurança, ferramentas avançadas de administração e suporte total à linguagem SQL. É muito usado em empresas que utilizam soluções Windows e sistemas críticos que exigem desempenho e confiabilidade.

MongoDB

MongoDB é um banco de dados NoSQL, que armazena dados em documentos JSON, oferecendo maior flexibilidade do que tabelas tradicionais. Ele é ideal para sistemas que lidam com grandes volumes de dados e precisam de escalabilidade e estrutura dinâmica, como aplicações modernas e serviços em nuvem.

MySQL

MySQL é um banco de dados relacional que organiza informações em tabelas e usa a linguagem SQL. É open-source, rápido e muito utilizado em sites, aplicações web e sistemas de médio a grande porte. É conhecido pela facilidade de uso e ampla compatibilidade.

Dicionário de dados e quais os tipos de dados de um banco de dados

O dicionário de dados é um documento que descreve cada campo do banco de dados: nome, tipo, tamanho, obrigatoriedade e significado. E os principais tipos de dados estão logo a baixo.

Texto (CHAR, VARCHAR)

São tipos usados para armazenar textos. CHAR guarda textos de tamanho fixo, ideal para códigos ou padrões; já VARCHAR armazena textos de tamanho variável, sendo mais eficiente para nomes, endereços e descrições.

Numérico (INT, FLOAT, DECIMAL)

Tipos numéricos armazenam valores matemáticos. INT guarda números inteiros, FLOAT armazena números com decimal porém com menor precisão, enquanto DECIMAL guarda números decimais com alta precisão, ideal para valores financeiros.

Lógico (BOOLEANO)

O tipo BOOLEANO armazena apenas dois valores: TRUE (verdadeiro) ou FALSE (falso). É usado em condições, estados e validações dentro do banco.

Data e horário (DATE, TIME, DATETIME)

Esses tipos representam tempo. DATE guarda datas, TIME guarda horários, e DATETIME combina os dois em um único valor, muito usado para registrar eventos e logs.

Binário (BLOB)

O tipo BLOB permite armazenar dados binários, como imagens, vídeos, documentos ou arquivos em geral. É útil quando o sistema precisa guardar mídias diretamente no banco de dados.

O que é uma modelagem de banco de dados e o modelo DER e MER

A modelagem de banco de dados é o processo de organizar e estruturar os dados antes de criá-los no sistema. O DER (Diagrama Entidade-Relacionamento) representa entidades, atributos e relacionamentos. O MER (Modelo Entidade-Relacionamento) é o modelo teórico que define como os dados se relacionam, servindo de base para o DER.
