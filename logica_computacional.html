Sobre lógica computacional

O conteúdo de lógica computacional aborda os fundamentos necessários para entender como programas são planejados e construídos. Inclui o uso de fluxogramas, algoritmos e Portugol para representar soluções de forma clara. Também apresenta conceitos como variáveis, constantes e tipos de dados, além de práticas com funções e vetores. Esses elementos ajudam o aluno a desenvolver raciocínio lógico e criar programas eficientes.

O que são fluxogramas e simbologia básica

Fluxogramas são representações gráficas de um processo ou algoritmo, mostrando passo a passo o caminho lógico de execução. Usam símbolos padronizados, como: oval (início/fim), retângulo (processos), Paralelogramo (Entrada/Saída), losango (decisões) e setas (fluxo). Eles ajudam a visualizar e planejar soluções antes da programação.

Oval (início/fim)

Indica o início ou o fim de um processo.

Retângulo (Processo)

Representa uma ação ou etapa do processo.

Losango (Decisão)

Mostra um ponto de decisão que geralmente tem como resultado um "sim" ou "não" e determina o próximo caminho no fluxograma.

Paralelogramo (Entrada/Saída)

Sinaliza quando dados entram ou saem do processo.

Seta (Linha de Fluxo)

Conecta os símbolos e indica a direção do fluxo do processo.

O que são algoritmos e onde são usados

Algoritmos são sequências de passos lógicos e ordenados criados para resolver um problema ou executar uma tarefa. São usados em programação, matemática, robótica, automação, softwares e até em atividades do cotidiano, como seguir uma receita ou instruções de montagem.

Portugol e Portugol Studio

Portugol é uma linguagem de pseudocódigo escrita em português, usada para ensinar lógica antes de aprender linguagens reais. O Portugol Studio é um software gratuito que permite escrever, executar e testar algoritmos em Portugol, facilitando o aprendizado com interface simples e comandos intuitivos.

O que são variáveis e constantes?

Variáveis são espaços na memória que armazenam valores que podem mudar durante a execução do programa. Já as constantes guardam valores fixos, que não podem ser alterados depois de declarados. Ambas são essenciais para organizar informações no algoritmo.

Quais os tipos de dados mais usados em algoritmos (portugol)? booleanos, caracter, cadeia (string), inteiro, real, vetor.

Booleano: verdadeiro/falso Caracter: armazena apenas 1 letra Cadeia (string): texto ou frases Inteiro: números sem decimal Real: números com decimal Vetor: estrutura que armazena vários valores do mesmo tipo

Converter quilograma-força por metro quadrado para libras-força por pé quadrado e vice versa

▼ 

programa
{
// converter 1 kgf/m2 para psf 
funcao real converterPsf(real kgf_m2){
	retorne 0.2048 * kgf_m2
}

// converter 1 psf para kgf/m2
funcao real converterKgf_m2(real psf){
	retorne 4.8824 * psf
}

funcao inicio()
{
	real kgf_m2 = 0.0, psf = 0.0, total = 0.0
	inteiro op = 0

	faca{

escreva("=================================== \n")

escreva(" 1 - converter kgf/m2 para psf \n")
escreva(" 2 - converter psf para kgf/m2 \n")
escreva(" Digite qual a sua opção: ")
leia(op)
escreva("=================================== \n")	

escolha(op){
	caso 0:
		escreva(" Saindo do Programa ... \n")
		pare
	caso 1:
		escreva(" convertendo kgf/m2 para psf \n")
		escreva(" Digite o valor em kgf/m2 : ")
		leia(kgf_m2)
		total = converterPsf(kgf_m2)
		escreva(" O valor total em psf é: ",total,"\n")
		pare
	caso 2:
		escreva(" convertendo de psf para kgf/m2 \n")
		escreva(" Digite o valor em psf : ")
		leia(psf)
		total = converterKgf_m2(psf)
		escreva(" O valor total em kgf/m2 é: ",total,"\n")
		pare
	caso contrario:
		escreva(" opção inválida, digite novamente! \n")
}
	}enquanto(op != 0)
	
	
}
}

calcular preço de desconto de produtos

▼ 

programa
{
funcao real calcularPreco(inteiro qtCimento, inteiro qtTinta, inteiro qtRejunte){
	retorne (qtCimento * 40.00) + (qtTinta * 80.00) + (qtRejunte * 15.00)
}

funcao real calcularDesconto(real precoSemD){
	real precoFinal = 0.0
	se(precoSemD >= 0.0 e precoSemD <= 100.0){
		precoFinal = precoSemD - (precoSemD *0.05)
		retorne precoFinal
	}senao se(precoSemD >100.0 e precoSemD <= 300.0){
		precoFinal = precoSemD - (precoSemD *0.07)
		retorne precoFinal
	}senao{
		precoFinal = precoSemD - (precoSemD *0.10)
		retorne precoFinal
	}
}

funcao inicio()
{
	// cimento = 40.00, tinta = 80.00, rejunte = 15.00
	real precoSemD = 0.0, precoComDeconto = 0.0
	inteiro qtCimento = 0, qtTinta = 0, qtRejunte = 0
	caracter op = 'S'

faca{
	escreva(" =================================================== \n")
	escreva("  Programa para Compra de Materiais de Construção    \n")
	escreva(" =================================================== \n")
	escreva("  'C' - para Compra \n")
	escreva("  'S' - para Sair  \n")
	escreva("    Digite sua opção: ")
	leia(op)
	escreva("\n")
	escreva(" ================================================ \n")

escolha(op){

		escreva("  Digite a quantidade de Cimento desejada: ")
		leia(qtCimento)

		escreva("  Digite a quantidade de Tinta desejada: ")
		leia(qtTinta)

	pare
	caso 'S':
		escreva(" Finalizando o programa \n")

		escreva(" Opção Inválida! Digite novamente! \n")
}

}enquanto(op != 'S')



	}

	se( num % 5 == 0){
		retorne num
	}senao{
		retorne 0
	}

	se( num % 7 == 0){
		retorne num
	}senao{
		retorne 0

{

Exemplos de scripts utilizando vetores no portugol

programa para identificar os números divisíveis por 3 e média deles

▼ 

programa
{
	
	funcao inicio()
	{
		inteiro vetor[9], num = 0
		real contador = 0.0, media = 0.0, soma = 0.0

		escreva(" programa para identificar os números divisíveis por 3 \n")
		escreva(" após identificar os divisíveis por 3 fazer a média aritmética \n")

		para(inteiro i = 0; i < 9; i++){
			num = sorteia(1,10)
			se(num % 3 == 0){
				vetor[i] = num
				contador = contador + 1
				soma = soma + vetor[i]	
			}senao{
				vetor[i] = 0
			}			
		}
		media = soma / contador
		escreva(" A média dos divisíveis por 3 é: ",media,"\n")
	}
}

programa para somar vetores, quando os elementos de mesmo índice do vetor que são pares, senão o valor é zero

▼ 

programa
{
	
	funcao inicio()
	{
		inteiro vetor1[5], vetor2[5], vetor3[5], num = 0
		real contador = 0.0, soma = 0.0, media = 0.0

		escreva("programa para somar vetores, quando os elementos  \n")
		escreva(" de mesmo índice do vetor que são pares, senão é zero \n")

		para(inteiro i = 0; i < 5; i++){
			num = sorteia(1,20)
			vetor1[i] = num
			num = sorteia(1,20)
			vetor2[i] = num 
		}
		para(inteiro i = 0; i < 5; i++){
			se((vetor1[i] % 2 == 0)e(vetor2[i] % 2 == 0)){
				vetor3[i] = vetor1[i] + vetor2[i]
			}senao{
				vetor3[i] = 0
			}
			escreva(" O vetor3 é: ",vetor3[i],"\n")
		}
	}
}
